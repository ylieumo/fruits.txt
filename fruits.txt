git init  - this command is use to make a folder into a git repo.
git status - use to check the stattus of the work inside the folder.
git add <filename> - use to stage work from the work area to the staging area.
git commit -m "message " - use to create history of our work.
git log - use to check the list of all our commit. 

===============================
git branch - allow us to list all our branches(the curent branch is marked with an asterix).
git branch <branch name> - allows us to create new branches in our repository.
git switch <branch name> - allows us to switch from one branch to another
